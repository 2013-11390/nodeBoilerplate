<h1 align="center">
  Node.js boilerplate
</h1>

Boilerplate for API server with node.js and GraphQL.
---

## Tech Stack
* Node.js, NPM, TypeScript, Babel, Prettier
* GraphQL
* PostgreSQL
* Jest

## Prerequisites
* Node.js
* PostgreSQL
* VSCode with ESlint, Prettier plugins

## Getting Started
```bash
$ git clone https://github.com/2013-11390/nodeBoilerplate.git
$ cd nodeBoilerplate
$ npm install
```
